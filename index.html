Welcome to BYOAI

This is a modular and customizable AI system inspired by Gentoo.

BYOAI allows users to build and integrate various AI models efficiently. Our goal is to democratize AI by making it accessible and easy to use for everyone.

With BYOAI, you can quickly and easily install and run several AI models, including:

  • GPT-2 Text Generation
  • BERT Named Entity Recognition (NER)
  • MobileNet Image Classification
  • TensorFlow Text Classification

Explore the GitHub repository for more details.
